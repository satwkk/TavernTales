# TAVERN TALES
A arcady cooking game set in medieval times where you harvest and cook foods for customers.

# GAMEPLAY ELEMENTS

### ENEMY Finite State Machine

FSM has multiple states
  * Idle
  * Wandering
  * ApproachShop
  * Eat
  * Leave
